ℹ Guide

GAP — Missing AIS data
A vessel had no AIS ping for more than 3 minutes between two consecutive positions. Possible reasons: transponder switched off, vessel left reception range, or signal interference.
Shown as: orange circles on the map.
JUMP — Impossible position leap
The implied speed between two consecutive pings exceeds 40 knots — physically impossible for a normal vessel. Caused by GPS errors, corrupted data, or duplicate MMSI numbers.
Shown as: red circles on the map.
